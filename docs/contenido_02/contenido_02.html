Contenido 2

Dominios, curvas de nivel y superficies con apoyo gráfico

En el contenido anterior aprendiste a mirar una función antes de calcular: identificaste qué entra, qué sale, si la salida es escalar o vectorial, y dónde puede estar definida. Ahora daremos el siguiente paso: convertir esa lectura en imagen. Cuando una función escalar depende de dos variables, como z = f(x, y), su dominio se lee como una región del plano, su gráfica puede representarse como una superficie en el espacio y sus curvas de nivel muestran dónde la función conserva el mismo valor. Estas tres miradas son complementarias. La región dice dónde vive la función; la superficie muestra cómo cambia la salida; las curvas de nivel permiten leer la función como un mapa. En ingeniería, esta habilidad es especialmente útil: muchas magnitudes no se entienden solo por una fórmula, sino por la forma en que cambian sobre una zona.

Ideas clave

  • El dominio de una función de dos variables suele ser una región, no solo una lista de números.
  • La superficie z = f(x, y) representa una salida escalar mediante altura.
  • Las curvas de nivel permiten leer una superficie desde arriba.
  • Una gráfica digital debe interpretarse y verificarse; no basta con insertarla.
  • Curvas de nivel más cercanas suelen sugerir cambios más rápidos, idea que prepara el gradiente.

Teoría guiada

Avanza en orden y usa cada imagen como apoyo para explicar con tus propias palabras.

1. Del dominio como conjunto al dominio como región

En cálculo de una variable, el dominio suele representarse mediante intervalos. Por ejemplo, una función puede estar definida para x >= 0 o para todos los reales excepto un punto. En funciones de dos variables, el dominio cambia de forma visual: ya no es una línea de valores permitidos, sino una región del plano. Esta diferencia es profunda porque obliga a leer desigualdades como zonas, fronteras, interiores, exteriores y bordes.

Si se tiene una función f(x, y), las entradas son pares ordenados. Cada par (x, y) corresponde a un punto del plano. El dominio es el conjunto de puntos que la función acepta. Si la función no tiene raíces pares, denominadores, logaritmos ni restricciones contextuales, el dominio puede ser todo el plano. Pero si aparece una condición como x^2 + y^2 <= 9, el dominio se convierte en un disco. Si aparece y >= x^2, el dominio es la región sobre una parábola. Si aparece x + y > 0, el dominio es un semiplano.

Leer el dominio como región ayuda a evitar errores posteriores. En integrales múltiples, por ejemplo, los límites de integración dependen de la región. En límites de varias variables, las trayectorias de aproximación se mueven dentro o cerca del dominio. En optimización, los máximos y mínimos pueden ocurrir dentro de la región o en su frontera. Por eso el dominio no es una formalidad: es el espacio donde el modelo vive.

2. Restricciones algebraicas frecuentes

Las restricciones algebraicas más comunes se reconocen desde cursos anteriores, pero en dos variables producen geometría. Una raíz cuadrada exige que el radicando sea mayor o igual que cero. Si f(x, y) = sqrt(x + y), el dominio se define por x + y >= 0. Esta desigualdad describe un semiplano. La frontera es la recta x + y = 0 y el dominio está de un lado de esa recta, incluyendo la frontera.

Un denominador no puede ser cero. Si g(x, y) = 1 / (x^2 + y^2 - 1), el dominio excluye los puntos que cumplen x^2 + y^2 - 1 = 0. La ecuación x^2 + y^2 = 1 representa una circunferencia. El dominio es todo el plano excepto esa circunferencia. En este caso no se elimina el interior ni el exterior completo, sino solo la frontera donde el denominador se anula.

Un logaritmo exige argumento positivo. Si h(x, y) = ln(4 - x^2 - y^2), el dominio cumple 4 - x^2 - y^2 > 0, es decir, x^2 + y^2 < 4. Aquí el borde no se incluye porque el logaritmo de cero no está definido. Es un disco abierto de radio 2. Esta diferencia entre <= y < es pequeña en símbolos, pero enorme en geometría: una incluye el borde; la otra lo excluye.

3. Dominio contextual en ingeniería

Además de las restricciones algebraicas, los modelos de ingeniería suelen tener restricciones contextuales. Una fórmula puede permitir todo el plano, pero la situación física no. Si una función T(x, y) describe temperatura en una placa rectangular, el dominio no es todo el plano, sino la región ocupada por la placa. Si una función P(x, y, z) describe presión en una muestra de roca, el dominio está limitado por la geometría de la muestra y por las unidades de medición.

El dominio contextual se construye desde el significado de las variables. Si x e y son coordenadas dentro de una sección circular, quizá deban cumplir x^2 + y^2 <= R^2. Si x representa distancia horizontal en un tramo de 100 metros, entonces 0 <= x <= 100. Si y representa profundidad positiva hacia abajo, se debe definir si y >= 0 o si se usa otra convención. La fórmula sola no siempre responde estas preguntas.

En la virtualización de este curso, se espera que el estudiante aprenda a escribir ambos dominios cuando sea necesario: el dominio algebraico y el dominio contextual. Esta doble lectura es una señal de pensamiento ingenieril: se respeta la matemática, pero también se respeta el fenómeno.

4. La gráfica de z = f(x, y) como superficie

Cuando f es una función escalar de dos variables, su gráfica está formada por puntos (x, y, z) tales que z = f(x, y). Las variables x e y ubican una entrada en el plano; la salida f(x, y) se representa como altura z. El resultado es una superficie en el espacio tridimensional. Esta superficie permite observar elevaciones, valles, simetrías, pendientes visuales y zonas de cambio rápido o lento.

Por ejemplo, f(x, y) = x^2 + y^2 produce una superficie en forma de paraboloide que se abre hacia arriba. En el origen, el valor es 0; al alejarse del origen, el valor aumenta. La fórmula y la superficie cuentan la misma historia en lenguajes distintos. La fórmula dice que la salida es la suma de dos cuadrados; la imagen muestra que el valor crece radialmente.

Otro ejemplo es f(x, y) = 9 - x^2 - y^2, una superficie que se abre hacia abajo. Su valor máximo ocurre en el origen y disminuye al alejarse. Si se restringe a los puntos donde la función es no negativa, se obtiene una especie de domo. Visualizar la superficie ayuda a anticipar comportamientos que luego se analizarán con derivadas y optimización.

5. Trazas: entender una superficie por cortes

Una superficie puede ser difícil de imaginar completa. Las trazas ayudan a estudiarla por cortes. Una traza se obtiene al intersectar la superficie con un plano. Si se fija x = a, se observa cómo cambia z con y sobre una línea vertical del dominio. Si se fija y = b, se observa cómo cambia z con x. Estas trazas convierten temporalmente un problema de dos variables en una curva de una variable.

Por ejemplo, para f(x, y) = x^2 + y^2, si se fija y = 0, queda z = x^2, una parábola. Si se fija x = 0, queda z = y^2, otra parábola. Estas trazas ayudan a entender por qué la superficie completa es un paraboloide. En una herramienta gráfica, mover cortes verticales permite ver familias de curvas que revelan la forma de la superficie.

Las trazas también preparan una idea importante: cuando más adelante calculemos una derivada parcial, estaremos observando el cambio en una dirección coordenada, algo muy parecido a mirar una traza y preguntarse por su pendiente en un punto.

6. Curvas de nivel: leer una superficie desde arriba

Una curva de nivel de una función f(x, y) es el conjunto de puntos del dominio donde la función toma un valor constante c. Se escribe f(x, y) = c. Si la superficie representa alturas, una curva de nivel une puntos que tienen la misma altura. Por eso los mapas topográficos usan curvas de nivel: permiten representar información tridimensional en un plano.

Para f(x, y) = x^2 + y^2, las curvas de nivel se obtienen resolviendo x^2 + y^2 = c. Si c > 0, estas curvas son circunferencias centradas en el origen con radio sqrt(c). Todas las entradas sobre una misma circunferencia producen el mismo valor de la función. Desde arriba, la superficie se ve como una familia de circunferencias.

Para f(x, y) = x + y, las curvas de nivel cumplen x + y = c, que son rectas paralelas. Esto muestra que no todas las curvas de nivel son circunferencias. La forma de las curvas depende de la función. Interpretarlas ayuda a reconocer simetrías, direcciones de cambio y zonas donde la función crece o decrece.

7. Separación entre dominio, superficie y curvas de nivel

Una dificultad común es mezclar tres objetos distintos: dominio, superficie y curvas de nivel. El dominio está en el plano de entrada; la superficie está en el espacio tridimensional; las curvas de nivel vuelven al plano de entrada, pero con información de la salida. Aunque se relacionan, no son lo mismo.

El dominio responde: ¿dónde puedo evaluar la función? La superficie responde: ¿qué altura o valor produce la función en cada punto? Las curvas de nivel responden: ¿dónde la función produce el mismo valor? Si el estudiante mantiene separadas estas preguntas, podrá leer mejor cualquier representación.

En herramientas digitales, la confusión puede aumentar porque una gráfica 3D se ve atractiva y parece explicarlo todo. Pero una superficie sin dominio correcto puede ser engañosa. Un mapa de contorno sin valores de nivel puede ser incompleto. Una captura sin interpretación no demuestra aprendizaje. Por eso cada representación debe acompañarse de explicación matemática.

8. Uso crítico de GeoGebra o herramienta equivalente

GeoGebra, Desmos 3D u otras herramientas pueden ayudar a visualizar funciones de dos variables. Sin embargo, la herramienta no reemplaza el razonamiento. Antes de graficar, el estudiante debe anticipar el dominio y el tipo de superficie. Después de graficar, debe verificar si la imagen coincide con la fórmula y con las restricciones.

Una buena práctica consiste en seguir una secuencia: primero escribir la función; segundo determinar el dominio; tercero identificar valores importantes; cuarto generar la superficie; quinto generar curvas de nivel o cortes; sexto escribir una interpretación. Si se invierte el orden y solo se pega una captura, se pierde la intención matemática del recurso.

La visualización debe responder preguntas. ¿Dónde crece la función? ¿Dónde parece tener un valor máximo o mínimo? ¿Qué forma tienen las curvas de nivel? ¿Hay simetría? ¿El dominio tiene borde incluido o excluido? ¿La representación usa unidades o escala adecuadas? Estas preguntas convierten la herramienta en apoyo de aprendizaje.

9. Relación con límites y derivadas parciales

Aunque este contenido no calcula todavía límites ni derivadas parciales en profundidad, prepara ambos temas. Los límites de funciones de dos variables exigen comprender que un punto puede ser alcanzado por muchas trayectorias dentro del dominio. Las curvas de nivel y los dominios ayudan a imaginar esas trayectorias.

Las derivadas parciales se entienden mejor cuando se han visto trazas. Derivar parcialmente respecto a x significa observar cómo cambia la salida cuando x varía y y se mantiene fijo. Visualmente, esto se parece a mirar una curva obtenida por un corte de la superficie. Por eso las trazas de este contenido serán retomadas en el siguiente.

El gradiente, que aparecerá más adelante, se relaciona con curvas de nivel porque apunta en la dirección de mayor crecimiento y es perpendicular a las curvas de nivel bajo condiciones adecuadas. Esa idea todavía no se formaliza aquí, pero se prepara al observar mapas de contorno: curvas juntas, curvas separadas, zonas altas y zonas bajas.

Método de trabajo

Método para analizar representaciones

  1. 1
    Identificar la función y su tipo.

    Confirmar que se trabaja con una función escalar de dos variables z = f(x, y).

  2. 2
    Determinar dominio algebraico.

    Revisar raíces, denominadores, logaritmos y cualquier condición que limite las entradas.

  3. 3
    Dibujar o describir la región del dominio.

    Traducir desigualdades a regiones: semiplanos, discos, exteriores, interiores, bandas o regiones delimitadas por curvas.

  4. 4
    Calcular valores de referencia.

    Evaluar la función en puntos simples para anticipar altura o comportamiento.

  5. 5
    Generar o bosquejar la superficie.

    Usar razonamiento o herramienta gráfica para visualizar z = f(x, y).

  6. 6
    Obtener curvas de nivel.

    Resolver f(x, y) = c para varios valores de c y describir la familia de curvas.

  7. 7
    Escribir interpretación.

    Explicar qué muestra cada representación y cómo se relacionan dominio, superficie y curvas de nivel.

Criterio de uso de herramienta

La herramienta gráfica debe usarse después de una lectura matemática inicial y antes de la interpretación final. Toda captura o visualización debe tener comentario propio.

Ejemplos comentados

Ejemplo 1. Paraboloide z = x^2 + y^2

Modelo: f(x, y) = x^2 + y^2

Analisis: El dominio es todo R^2 porque no hay restricciones algebraicas. La superficie es un paraboloide que se abre hacia arriba. Las curvas de nivel cumplen x^2 + y^2 = c, por lo que son circunferencias para c > 0. El origen produce el valor mínimo 0.

Aprendizaje: Una misma fórmula se puede leer como región total, superficie y familia de curvas de nivel.

Ejemplo 2. Plano z = x + y

Modelo: f(x, y) = x + y

Analisis: El dominio es todo R^2. La gráfica es un plano. Las curvas de nivel cumplen x + y = c, una familia de rectas paralelas. Si c aumenta, las rectas se desplazan manteniendo la misma inclinación.

Aprendizaje: Las curvas de nivel de una función lineal son rectas paralelas.

Ejemplo 3. Domo restringido por raíz

Modelo: f(x, y) = sqrt(9 - x^2 - y^2)

Analisis: El dominio exige x^2 + y^2 <= 9. La superficie corresponde a la semiesfera superior de radio 3. Las curvas de nivel cumplen sqrt(9 - x^2 - y^2) = c, lo cual lleva a x^2 + y^2 = 9 - c^2 para 0 <= c <= 3.

Aprendizaje: La restricción del dominio y la forma de las curvas de nivel están conectadas.

Problemas resueltos paso a paso

Problema resuelto 1. Dominio y curvas de nivel de una raíz

Nivel: Universitario introductorio

Contexto: Una magnitud superficial disminuye radialmente desde un punto central.

Enunciado: Sea f(x, y) = sqrt(25 - x^2 - y^2). Determine el dominio, describa la superficie y obtenga las curvas de nivel principales.

  1. 1
    Identificar restricción.

    La raíz cuadrada exige 25 - x^2 - y^2 >= 0.

  2. 2
    Despejar condición del dominio.

    La condición equivale a x^2 + y^2 <= 25. Esto representa un disco cerrado de radio 5 centrado en el origen.

  3. 3
    Describir superficie.

    Como z = sqrt(25 - x^2 - y^2), se tiene z >= 0. Al elevar al cuadrado, z^2 = 25 - x^2 - y^2, es decir, x^2 + y^2 + z^2 = 25. La superficie es la semiesfera superior de radio 5.

  4. 4
    Obtener curvas de nivel.

    Para f(x, y) = c, se cumple sqrt(25 - x^2 - y^2) = c. Entonces x^2 + y^2 = 25 - c^2. Las curvas son circunferencias para 0 <= c <= 5.

  6. 5
    Interpretar niveles.

    Cuando c = 5, la curva de nivel se reduce al punto (0, 0). Cuando c = 0, la curva es el borde del dominio, x^2 + y^2 = 25. Los niveles intermedios son circunferencias.

Resultado: Dominio: x^2 + y^2 <= 25. Superficie: semiesfera superior de radio 5. Curvas de nivel: x^2 + y^2 = 25 - c^2, con 0 <= c <= 5.

Verificacion: En el origen f(0,0)=5, valor máximo. En el borde del disco, f=0. Fuera del disco, la función no está definida en reales.

Transferencia: Esta lectura será útil para integrales en regiones circulares y para interpretar curvas de nivel como mapas.

Problema resuelto 2. Región excluida por un denominador

Nivel: Universitario introductorio

Contexto: Un modelo racional describe una respuesta que no está definida sobre una frontera crítica.

Enunciado: Sea g(x, y) = 1 / (x^2 + y^2 - 4). Determine el dominio y explique qué región se excluye.

  1. 1
    Identificar restricción.

    El denominador no puede ser cero: x^2 + y^2 - 4 != 0.

  2. 2
    Resolver condición prohibida.

    La condición x^2 + y^2 - 4 = 0 equivale a x^2 + y^2 = 4, una circunferencia de radio 2.

  3. 3
    Describir dominio.

    El dominio es todo el plano excepto la circunferencia de radio 2. El interior del círculo sí pertenece al dominio, y el exterior también; solo se excluye el borde crítico.

  4. 4
    Interpretar comportamiento.

    Cerca de la circunferencia excluida, el denominador se acerca a cero y la función puede crecer en magnitud. Esto advierte que la gráfica tendrá comportamiento abrupto cerca de esa frontera.

Resultado: Dominio: R^2 menos la circunferencia x^2 + y^2 = 4.

Verificacion: El punto (0,0) pertenece al dominio porque el denominador vale -4. El punto (2,0) no pertenece porque el denominador vale 0.

Transferencia: Distinguir entre excluir una frontera y excluir una región completa será importante al plantear dominios e integrales.

Problema resuelto 3. Curvas de nivel de una función lineal

Nivel: Universitario introductorio

Contexto: Una magnitud cambia linealmente sobre una región plana.

Enunciado: Sea h(x, y) = 2x - y. Describa su dominio, superficie y curvas de nivel.

  1. 1
    Dominio.

    No hay raíces, divisiones ni logaritmos. El dominio algebraico es todo R^2.

  2. 2
    Superficie.

    La gráfica z = 2x - y es un plano. Aumenta cuando x aumenta y disminuye cuando y aumenta.

  3. 3
    Curvas de nivel.

    Para h(x, y) = c, se tiene 2x - y = c. Al despejar, y = 2x - c. Las curvas de nivel son rectas paralelas de pendiente 2.

  4. 4
    Interpretación.

    Cada recta reúne puntos donde la magnitud tiene el mismo valor. Cambiar c desplaza la recta sin cambiar su pendiente.

Resultado: Dominio: R^2. Superficie: plano. Curvas de nivel: y = 2x - c.

Verificacion: Si c = 0, la curva es y = 2x. Si c = 1, la curva es y = 2x - 1. Son paralelas.

Transferencia: Las curvas de nivel lineales ayudan a visualizar cambios constantes y preparan la lectura del gradiente.

Cierre

Una función escalar de dos variables puede leerse desde tres representaciones: dominio, superficie y curvas de nivel. El dominio muestra dónde está definida; la superficie muestra cómo varía la salida; las curvas de nivel muestran dónde la salida permanece constante.

Estas representaciones preparan el estudio de límites, continuidad, derivadas parciales, gradiente, optimización e integración sobre regiones.

Pregunta para pensar: ¿Qué información se pierde si solo miras la fórmula y no observas el dominio, la superficie o las curvas de nivel?

En el Contenido 3 estudiarás cómo se comportan estas funciones cerca de un punto y cómo medir cambios mediante derivadas parciales.
